# E2E Testing

**End-to-End (E2E)** testing is utilized to ensure the application flow from the start to the end is correct.

While it's not required to write an e2e test at the time of this writing, e2e test is highly encouraged to have whenever is possible.

In order to do an e2e test, we use [cypress](https://www.cypress.io/) as our primary e2e testing tools. It's a purely JS based, fast, and reliable.
